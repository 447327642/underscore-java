Underscore-java by javadev

Underscore-java is a java port of the popular Underscore.js library. In addition to porting Underscore's functionality, Underscore.php includes matching unit tests. Underscore-java requires java 1.6 or greater.

Underscore-java is hosted on GitHub and open sourced under the MIT license.

Download

You can download this project in either zip or tar formats.
